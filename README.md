# Sparv

Sparv is a interpreted dynamically typed language. It aims to be a all-purpose
programming language with inspiration from many different language. If you familiar with any c-style
language you probably will feel at home.

## Installation

The easiset way to install sparv is to build [Rust](https://www.rust-lang.org/)

```
    git clone https://github.com/lhedeby/sparv
    cd sparv
    cargo build
```

Then place the binary in your PATH

## Getting started

With the binary in your path it's as simple as ``sparv <your file>``

For tips on getting started checkout [getting started](https://github.com/lhedeby/sparv/blob/main/docs/getting-started.md)
